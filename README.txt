Run Spring boot application by running main method of ExchangerateApplication

View Current date exchange rate by hitting URL on web browser
http://localhost:8080/exchangerates/
http://localhost:8080/exchangerates/2020-04-01

To throw any error provide any invalid date
http://localhost:8080/exchangerates/invalid
http://localhost:8080/exchangerates/2020-20-20

To tun test cases, go to ExchangeRateWelcomeControllerTest.java and run.
